# Code of Conduct

MarkDownshift is intended to be a welcoming, practical community project.

Be respectful, constructive, and patient. Focus criticism on ideas and code, not people. Harassment, discrimination, threats, deliberate intimidation, and publication of another person's private information are not acceptable.

Project maintainers may edit or remove contributions and restrict participation when necessary to protect the community. To report conduct concerns privately, contact the project owner through the email address listed on the GitHub profile for **Jean-MarcFrayret**.
