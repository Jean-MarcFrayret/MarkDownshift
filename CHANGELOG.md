# MarkDownshift Changelog

## [1.3.0] - 2026-07-22

- Added synchronized editor and preview zoom controls
- Added `⌘−`, `⌘0`, and `⌘+` zoom shortcuts

## [1.2.0] - 2026-07-21

- Added native multi-document windows
- Fixed opening Markdown files from Finder
- Added a visible Close File button and standard `⌘W` handling

## [1.1.0] - 2026-07-21

- Added block-aware Markdown preview rendering
- Added the gold and hot-rod-red application icon

## [1.0.0] - 2026-07-21

- Initial release with direct Markdown editing, preview, formatting controls, and native save panels
